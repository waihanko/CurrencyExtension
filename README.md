## Download
Add it in your root build.gradle at the end of repositories:
```bash
allprojects 
{ repositories 
    {
	maven { url 'https://jitpack.io' }
    }
}
  ```

Add the dependency
```bash
dependencies
{
	implementation 'com.github.waihanko:MCFormatter:0.0.1'
}
  ```
